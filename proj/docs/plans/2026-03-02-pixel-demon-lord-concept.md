# 像素魔王地下城 - 遊戲概念

> 非對稱對戰：玩家扮演像素魔王，建造地下城、引誘勇者、收割靈魂；同時要避免被生還勇者留下傷口，並可將強力勇者送往對手地下城。

---

## 核心身份

**玩家 = 像素魔王**：建造地下城，引誘勇者進入，然後消滅他們。

---

## 勝負條件


| 條件                 | 結果       |
| ------------------ | -------- |
| 收集到 **10 個靈魂**     | 該玩家獲勝    |
| 城鎮上或勇者牌庫中 **沒有勇者** | 遊戲結束（結算） |
| 魔王被勇者留下 **傷口** 致死  | 該魔王出局    |


---

## 核心資源

### 靈魂（Souls）

- **來源**：勇者斃命於地下城內
- **用途**：勝利條件（10 個 = 贏）

### 傷口（Wounds）

- **來源**：勇者生還、逃出地下城
- **效果**：魔王受傷，累積可致死

---

## 核心機制

### 1. 吸引勇者

- 勇者不會去沒有寶藏的地方
- 魔王需要**佈置誘餌**（寶藏、陷阱組合）吸引勇者
- 吸引力 vs 危險度：平衡「吸引過來」與「消滅他們」

### 2. 消滅勇者

- 地下城內設陷阱、怪物、機關
- 勇者進入 → 觸發戰鬥/陷阱 → 死亡則收割靈魂

### 3. 反利用：送勇者給對手

- 可將**強大的勇者**送往**其他魔王的地下城**
- 讓對手「受傷」從而死亡
- **風險**：勇者在城鎮會慢慢變強，過度內鬥會讓勇者變得更強，反噬所有人

---

## 策略張力


| 策略       | 收益    | 風險            |
| -------- | ----- | ------------- |
| 佈置陷阱消滅勇者 | 收割靈魂  | 吸引力不足則沒勇者來    |
| 提高吸引力    | 勇者來得多 | 可能打不過，生還者留下傷口 |
| 送強勇者給對手  | 對手受傷  | 勇者變強、城鎮勇者變少   |


---

## 遊戲流程（推測）

1. **建造階段**：魔王佈置地下城（陷阱、寶藏、怪物）
2. **吸引階段**：勇者根據吸引力選擇進入哪個地下城
3. **戰鬥階段**：勇者在地下城內遭遇陷阱/怪物
4. **結算階段**：死亡 → 靈魂；生還 → 傷口
5. **城鎮階段**：勇者變強（若未消滅）
6. 重複直到達成勝利條件

---

## 與現有項目的關係

- **可獨立**：全新概念，與傳奇地形項目無直接關聯
- **可融合**：若沿用傳奇 .map 結構，可將「地下城」視為地圖塊，勇者為「怪物」的反向（玩家對抗的 NPC）
- **技術棧**：Unity、2D/3D 像素均可。可破壞、可建造與現有 BlockController 有共通點
