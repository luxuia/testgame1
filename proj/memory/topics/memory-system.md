# 主题：记忆体系 (Memory System)

## 概述

本项目采用「热-温-冷」三层记忆结构，用于知识沉淀与检索。

## 三层结构

### L1 当日原始层（热）

- **文件**：`memory/YYYY-MM-DD.md`
- **作用**：流水账，记录当天决策、执行、报错、结果
- **特点**：全量、不筛选、写完就走

### L2 主题提炼层（温）

- **文件**：`memory/topics/*.md`
- **作用**：把 daily 里的高价值内容按主题沉淀（如投资、PAI、安全、教训）
- **特点**：结构化、可复用、可检索

### L3 长期记忆层（冷）

- **文件**：`MEMORY.md`
- **作用**：只保留长期有效的 P0/P1 级原则、事实、关键结论
- **特点**：极简、稳定、低噪音

## 使用流程

1. 每日在 `memory/YYYY-MM-DD.md` 记录当日流水
2. 定期从 daily 中提炼高价值内容，写入 `memory/topics/<主题>.md`
3. 将跨项目适用的原则升级到 `MEMORY.md`
